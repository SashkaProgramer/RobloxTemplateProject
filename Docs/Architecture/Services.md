# Services

## RoundService

- Match lifecycle
- Phase transitions
- Timer control
- Extraction opening and closure
- Match result resolution

## WorldService

- Train setup and car registry
- Door states
- Lighting and power state
- Interaction object lookup

## LootService

- Loot definitions and spawn tables
- Spawn selection
- Pickup/drop validation
- Secured loot value calculation

## PlayerStateService

- Alive/dead/extracted states
- Current car tracking
- Inventory state
- Temporary round flags

## InfectedService

- Threat spawning
- Noise reaction
- Targeting and pressure spikes

## SabotageService

- Locking doors
- Cutting power
- Triggering noise
- Stealing or moving key resources

## RewardService

- Currency rewards
- Extraction rewards
- Match summary data
- Persistent stat updates
