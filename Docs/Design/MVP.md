# MVP

## MVP Goal

Ship one complete, tense, replayable train raid with enough systems to prove the game loop and support first monetization layers later.

## MVP Scope

- One train route
- One match mode
- `6-8` wagon segments
- One main PvE threat
- Sabotage-based player conflict
- Limited extraction slots
- Run-based loot persistence

## MVP Success Criteria

- A full match can start, progress, escalate, and resolve reliably.
- Players understand the goal without a large tutorial.
- Rare loot changes behavior.
- Extraction endings create memorable outcomes.

## Out Of Scope

- Large content variety
- Multiple enemy families
- Deep player progression
- Full cosmetic catalog
- Rich pre-match lobby systems
