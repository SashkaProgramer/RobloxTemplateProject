# Decisions

## Confirmed Decisions

- Genre direction: round-based social survival / extraction-lite
- Fantasy shell: cursed night train
- Tone: dark, anxious, atmospheric, with chaos in spikes
- PvP style: sabotage-driven rather than direct combat-first PvP
- Extraction rule: not all players can escape
- Rare loot: included in MVP
- Visual direction: grounded and semi-realistic rather than cartoonish

## Working Assumptions

- MVP should focus on one train route and one match mode
- Match length should stay around `8-10 minutes`
- Strongest social tension should happen late in the run
